#Text Classification Engine

An abstraction of some techniques for distinguishing two texts,
based on typical Linguistic techniques.

This engine lets you run a few standard analyses of any (Unicode) corpus,
or create your own.

##Built in techniques:

- vowels per word
- consonants per word
- etc
